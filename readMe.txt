## Проект "Список дел"

### ОСНОВНАЯ ИДЕЯ:

Основной идеей этого веб-приложения является возможность пользователей вводить текстовые задачи, 
которые затем отображаются в виде списка на странице. Это позволяет пользователям удобно отслеживать свои задачи 
и помогает им быть более организованными.

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

1. **HTML-страница:** содержит заголовок, текстовое поле для ввода задач, кнопку для добавления задачи и список задач. 
Заголовок страницы динамически меняется в зависимости от имени пользователя.

2. **JavaScript (script.js):** используется для управления взаимодействием пользователя с приложением. 
Он обрабатывает ввод текста, добавление задач в список, подсчет введенных задач и вывод сообщения, 
если пользователь превышает лимит в 5 задач.

3. **CSS-стили:** Для улучшения визуального восприятия приложения, применены CSS-стили и классы к элементам, 
чтобы сделать интерфейс более привлекательным.

4. **Внешние ресурсы:** Для улучшения внешнего вида используются внешние шрифты, полученные из Google Fonts.

### ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

- **HTML:** для разметки веб-страницы.
- **JavaScript:** для создания интерактивности и управления поведением приложения.
- **CSS:** для стилизации и улучшения пользовательского интерфейса.

### ОСОБЕННОСТИ ПРОЕКТА:

- При открытии страницы, приложение запрашивает имя пользователя с помощью функции `prompt` и динамически изменяет заголовок страницы.
- Введенные задачи добавляются в список без удаления предыдущих задач.
- Пользователь ограничен вводом не более 5 задач, и если он попытается ввести больше, приложение выведет сообщение о максимальном лимите.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ:

1. Склонируйте репозиторий на свой компьютер.
2. Откройте `index.html` в вашем веб-браузере.
